## 📖 Project Overview

- POST - Create token
- GET - list of booking IDs
- GET - booking IDs useing query
- GET - detail of booking JSON
- GET - detail of booking XML
- POST - create booking JSON
- POST - create booking XML
- PUT - update booking
- PATCH - partial update booking
- DEL - delete booking
